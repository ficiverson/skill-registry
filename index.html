Welcome to the Skill Registry. This entire index is generated and maintained automatically using the Skills Forge library, crafted by Fernando Souto.

The Skill Forge framework is a distribution ecosystem for AI capabilities. It allows engineers to package specialized knowledge, architectural patterns, and technical workflows into deterministic .skillpack artifacts. By using a standard GitHub repository as a decentralized registry, teams can version, discover, and share these skills securely via SHA256 checksums. Once installed via the CLI, these packs empower AI agents like Claude Code with rich, context-aware expertise specifically tailored to your technological stack.
